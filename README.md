## 👋 Hey, I'm Ibrahim

**AI Engineer · Team Lead** at **MTBC / CareCloud**, based in **Islamabad**. I build production **GenAI systems** — multi-layered RAG pipelines, agentic workflows (LangGraph · CrewAI · MCP), and LLM-powered services that ship on **FastAPI · Docker · Kubernetes**. I care about the unglamorous parts — reliable retrieval, evaluation, observability — because that's what separates a working prototype from something that earns trust in production.

## 🚀 Featured — AI Medical Coding Assistant

> Production GenAI at **MTBC / CareCloud**. Multi-layered RAG over **Qdrant**, customized per practice and specialty, with Gemini grounding for up-to-date payer rules (MIPS / LCD / NCD / NCCI). Wrapped in FastAPI and shipped via Azure DevOps CI/CD on Kubernetes — helped reduce per-record medical coding time from **~20 min → ~5 min**.

## 🛠️ Projects

- 🛡️ **[Sentinel AI](https://github.com/IbrahimAbid0)** — Turns cheap CCTV feeds into a smart surveillance system without an expensive DVR. Face detection (MTCNN), recognition (InceptionNet), liveness detection, and an AI caller agent (DeepGram STT/TTS) for emergencies.
- 🧭 **[Agentic Travel Planner](https://github.com/IbrahimAbid0)** — LangGraph multi-agent system (Flight · Hotel · Restaurant) coordinated by a planner agent. Tools wired via **Model Context Protocol (MCP)**.
- 🌦️ **[MLOps Weather Pipeline](https://github.com/IbrahimAbid0/MLops-A-01)** — End-to-end MLOps: DVC + MLflow + Docker + Kubernetes (Minikube) + CI/CD for real-time temperature prediction.
- 🧠 **[Multi-Modal Meme Sentiment](https://github.com/IbrahimAbid0)** — PyTorch late-fusion model (Word2Vec + BERT + ResNet50). 82% binary accuracy on a notoriously noisy domain.
- 🏠 **[House Price Prediction](https://github.com/IbrahimAbid0/House-Price-Prediction)** — End-to-end ML deployment pipeline demonstrating the full MLOps lifecycle.
